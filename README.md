2. addCartForm.html為加入購物車之前端程式

11. myCart為"我的購物車"

12. removeFromCart.py為將商品拿出購物車之後端程式

13. ShoppingCart.png為我嘗試做的這次Shopping Cart的使用者情境(User Case)，不一定正確。
